# PDF Generator

Простое Flask приложение для создания PDF документов с пользовательскими полями ввода.

## Возможности

- Создание PDF документов с настраиваемыми полями
- Красивый веб-интерфейс с Bootstrap
- Поддержка таблиц и форматирования
- Автосохранение данных формы
- Валидация входных данных
- Адаптивный дизайн

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Использование

1. Заполните необходимые поля в веб-форме
2. Обязательные поля: Заголовок и Содержание
3. Нажмите "Создать PDF" для генерации документа
4. Файл автоматически скачается в браузере

## Структура проекта

- `app.py` - основное приложение Flask
- `templates/` - HTML шаблоны
- `requirements.txt` - зависимости Python
- `temp_pdfs/` - временные PDF файлы (создается автоматически)

## Возможности PDF

- Форматированный текст с заголовками
- Таблицы с дополнительными данными
- Дата создания документа
- Подпись автора
- Профессиональное оформление

## Технические детали

- Flask для веб-сервера
- ReportLab для генерации PDF
- Bootstrap 5 для интерфейса
- Автоматическая очистка временных файлов
- Поддержка русского языка
