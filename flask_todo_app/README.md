# Flask Todo App

Простое веб-приложение для управления задачами, построенное на Flask.

## Возможности

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Редактирование существующих задач
- ✅ Удаление задач
- ✅ Отображение даты создания
- ✅ Интуитивно понятный интерфейс

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.7+

2. Установите зависимости:
 ```bash
 pip install -r requirements.txt
 ```

3. Запустите приложение:
 ```bash
 python app.py
 ```

4. Откройте браузер и перейдите по адресу: http://127.0.0.1:5000

## Структура проекта

```
flask_todo_app/
├── app.py # Основной файл приложения
├── requirements.txt # Зависимости
├── templates/
│ ├── index.html # Главная страница
│ └── edit.html # Страница редактирования
├── todo.db # SQLite база данных (создается автоматически)
└── README.md # Документация
```

## Технологии

- **Backend**: Flask (Python)
- **Database**: SQLite + SQLAlchemy ORM
- **Frontend**: HTML, CSS
- **Styling**: Встроенные CSS стили

## API Endpoints

- `GET /` - Главная страница со списком задач
- `POST /add` - Добавление новой задачи
- `GET /complete/<id>` - Переключение статуса выполнения задачи
- `GET /delete/<id>` - Удаление задачи
- `GET/POST /edit/<id>` - Редактирование задачи

## Особенности

- Автоматическое создание базы данных при первом запуске
- Responsive дизайн
- Подтверждение удаления задач
- Визуальное отображение выполненных задач
- Сортировка задач по дате создания (новые сверху)
