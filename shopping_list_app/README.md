# Shopping List App

Простое и удобное веб-приложение для управления списком покупок, созданное на Flask.

## Возможности

- ✅ Добавление товаров с указанием количества и категории
- ✅ Отметка купленных товаров
- ✅ Удаление отдельных товаров
- ✅ Массовое удаление купленных товаров
- ✅ Категоризация товаров
- ✅ Статистика покупок
- ✅ Адаптивный дизайн

## Технологии

- **Backend**: Flask, SQLAlchemy
- **Frontend**: HTML, CSS, Bootstrap 5, Font Awesome
- **База данных**: SQLite

## Установка и запуск

1. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate # Linux/Mac
venv\Scripts\activate # Windows
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python app.py
```

4. Откройте браузер и перейдите по адресу: http://127.0.0.1:5000

## Использование

1. **Добавление товара**: Заполните форму в верхней части страницы
2. **Отметка как купленного**: Нажмите на квадратик рядом с товаром
3. **Удаление товара**: Нажмите кнопку с корзиной
4. **Очистка купленных**: Используйте кнопку "Очистить выполненные"

## Категории товаров

- Продукты

- Хлеб
