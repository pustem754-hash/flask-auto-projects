# Flask Калькулятор процентов

Веб-приложение для расчета различных типов процентов с интуитивным интерфейсом.

## Возможности

- **Процент от числа** - рассчитать X% от числа
- **Какой процент** - узнать, какой процент составляет одно число от другого
- **Процентное изменение** - рассчитать изменение в процентах между двумя значениями
- **Прибавить процент** - увеличить число на определенный процент
- **Отнять процент** - уменьшить число на определенный процент

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте в браузере:
```
http://localhost:5000
```

## Технологии

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **API**: RESTful JSON API

## Примеры использования

- Расчет скидок и наценок
- Финансовые вычисления
- Статистический анализ
- Образовательные цели

## API

### POST /calculate

Выполняет расчет процентов.

**Параметры:**
- `type`: тип расчета
- Дополнительные параметры в зависимости от типа

**Ответ:**
```json
{
 "result": 25.0,
 "formula": "25% от 100 = 25.0"
}
```
