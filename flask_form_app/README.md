# Flask Form Application

Простое Flask веб-приложение с формой обратной связи.

## Возможности

- Форма с валидацией полей
- Поля: имя, email, тема, сообщение
- Красивый интерфейс с CSS стилизацией
- Обработка ошибок валидации
- Страница успешной отправки
- Flash-сообщения

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Технологии

- **Python 3.x**
- **Flask 3.0.0** - веб-фреймворк
- **Flask-WTF** - работа с формами
- **WTForms** - валидация форм
- **HTML/CSS** - интерфейс

## Функционал

### Валидация формы:
- Имя: обязательное поле, 2-50 символов
- Email: обязательное поле, корректный email адрес
- Тема: выбор из предустановленных вариантов
- Сообщение: обязательное поле, 10-500 символов

### Безопасность:
- CSRF защита через Flask-WTF
- Валидация на стороне сервера
- Экранирование пользовательского ввода

## Настройка

Для продакшена измените SECRET_KEY в app.py на уникальный ключ:
```python
app.config['SECRET_KEY'] = 'ваш-уникальный-секретный-ключ'
```
