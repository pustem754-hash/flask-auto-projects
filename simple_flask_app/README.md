# Simple Flask App

Простое Flask приложение с одной страницей и красивым дизайном.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Структура проекта

- `app.py` - основной файл приложения
- `templates/index.html` - HTML шаблон
- `static/style.css` - стили CSS
- `requirements.txt` - зависимости Python

## Функциональность

- Главная страница `/`
- Страница "О проекте" `/about`
- Адаптивный дизайн
- Красивые градиенты и анимации
