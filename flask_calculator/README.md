# Flask Калькулятор

Веб-приложение калькулятора на Flask с поддержкой базовых математических операций и функций.

## Возможности

- Базовые операции: +, -, *, /
- Тригонометрические функции: sin, cos, tan
- Квадратный корень (sqrt)
- Логарифм (log)
- Работа со скобками
- История вычислений
- Поддержка клавиатуры
- Адаптивный дизайн

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Использование

- Используйте кнопки интерфейса или клавиатуру для ввода выражений
- Нажмите "=" или Enter для вычисления
- Используйте "C" или Escape для очистки
- Backspace для удаления последнего символа
- История вычислений отображается внизу

## Поддерживаемые функции

- `sin(x)` - синус
- `cos(x)` - косинус 
- `tan(x)` - тангенс
- `sqrt(x)` - квадратный корень
- `log(x)` - натуральный логарифм
- `π` - число пи
