# Калькулятор Скидок

## Описание
Веб-приложение на Flask для расчета скидок с интуитивным пользовательским интерфейсом.

## Возможности
- ✅ Расчет скидки для одного товара
- ✅ Массовый расчет для нескольких товаров
- ✅ Красивый адаптивный интерфейс
- ✅ Валидация данных
- ✅ REST API
- ✅ Обработка ошибок

## Функции

### Одиночный калькулятор
- Ввод первоначальной цены и процента скидки
- Автоматический расчет суммы скидки и финальной цены
- Отображение всех параметров

### Массовый калькулятор
- Добавление неограниченного количества товаров
- Индивидуальные скидки для каждого товара
- Подсчет общей экономии и итоговой стоимости
- Расчет среднего процента скидки

## API Endpoints

### POST /calculate
Расчет скидки для одного товара:
```json
{
 "original_price": 1000,
 "discount_percent": 15
}
```

Расчет для нескольких товаров:
```json
{
 "items": [
 {"name": "Товар 1", "price": 1000, "discount": 15},
 {"name": "Товар 2", "price": 500, "discount": 10}
 ]
}
```

### GET /api/health
Проверка состояния сервиса

## Технологии
- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Стилизация**: Custom CSS с градиентами
- **API**: RESTful JSON API

## Установка и запуск

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Запуск приложения:
```bash
python app.py
```

3. Откройте в браузере: http://localhost:5000

## Примеры использования

### Расчет одной скидки
1. Введите цену товара: 1000
2. Введите процент скидки: 15
3. Получите результат:
 - Скидка: 150 ₽
 - Итоговая цена: 850 ₽

### Массовый расчет
1. Добавьте товары с разными ценами и скидками
2. Получите детальный отчет по каждому товару
3. Увидите общую статистику и экономию
