# Flask Notes App

Простое веб-приложение для создания заметок с системой тегов.

## Функциональность

- ✅ Создание, редактирование и удаление заметок
- ✅ Система тегов для организации заметок
- ✅ Поиск по заголовку и содержимому
- ✅ Фильтрация по тегам
- ✅ Адаптивный дизайн с Bootstrap
- ✅ Управление тегами

## Установка и запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000

## Использование

1. **Создание заметки**: Нажмите "Новая заметка" и заполните форму
2. **Добавление тегов**: В поле "Теги" введите теги через запятую
3. **Поиск**: Используйте строку поиска для поиска по содержимому
4. **Фильтрация**: Выберите тег в выпадающем списке для фильтрации
5. **Управление тегами**: Перейдите в раздел "Теги" для просмотра статистики

## Структура проекта

```
flask_notes_app/
├── app.py # Основное приложение Flask
├── requirements.txt # Зависимости
├── notes.db # База данных SQLite (создается автоматически)
└── templates/ # HTML шаблоны
 ├── base.html # Базовый шаблон
 ├── index.html # Главная страница
 ├── note_form.html # Форма создания/редактирования
 ├── view_note.html # Просмотр заметки
 └── tags.html # Управление тегами
```

## Технологии

- **Flask** - веб-фреймворк
- **SQLAlchemy** - ORM для работы с базой данных
- **WTForms** - обработка форм
- **Bootstrap 5** - CSS фреймворк
- **Font Awesome** - иконки
- **SQLite** - база данных
