# База домофонов

Веб-приложение для управления базой данных домофонов с кодами доступа.

## Функции

- ✅ Добавление домофонов с адресом, подъездом, кодом и заметками
- ✅ Просмотр всех домофонов в удобном интерфейсе
- ✅ Редактирование информации о домофонах
- ✅ Удаление домофонов
- ✅ Поиск по адресу и коду
- ✅ Адаптивный дизайн для мобильных устройств

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу: http://localhost:5000

## Использование

1. **Добавление домофона**: Нажмите "Добавить домофон" и заполните форму
2. **Поиск**: Используйте поле поиска для поиска по адресу или коду
3. **Редактирование**: Нажмите "Изменить" на карточке домофона
4. **Удаление**: Нажмите "Удалить" на карточке домофона

## Структура данных

Каждый домофон содержит:
- **Адрес** (обязательно): Полный адрес здания
- **Подъезд** (опционально): Номер подъезда
- **Код доступа** (обязательно): Код для входа
- **Заметки** (опционально): Дополнительная информация

## Технологии

- **Backend**: Flask (Python)
- **Database**: SQLite
- **Frontend**: Bootstrap 5, Font Awesome
- **Forms**: Flask-WTF, WTForms

## Безопасность

Перед использованием в продакшене:
1. Измените SECRET_KEY в app.py
2. Используйте более надежную базу данных (PostgreSQL, MySQL)
3. Добавьте аутентификацию пользователей
4. Настройте HTTPS
